# Tagging Consistency Rule

- ONLY use tags from the controlled vocabulary defined in CLAUDE.md
- Maximum 6 tags per episode
- At least 1 tag MUST be a domain tag (ai, neuroscience, startups, etc.)
- Tags must reflect actual episode content, not assumed topics
- Novelty score REQUIRES a Memory MCP query against existing concepts — never guess
- Relevance score is against user interests listed in CLAUDE.md
- Complexity score reflects depth of technical content (1=casual, 10=PhD-level)

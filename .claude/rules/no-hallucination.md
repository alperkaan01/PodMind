# No Hallucination Rule

Every insight, quote, person, book, and concept extracted MUST be explicitly
present in the transcript or show notes. No exceptions.

- NEVER infer, assume, or add information from training data
- NEVER fill gaps with plausible-sounding but unverified claims
- If uncertain whether something was said, omit it entirely
- Quotes MUST be verbatim — never paraphrase and mark as a quote
- Speaker attribution must come from the transcript, not guessed
- Book/paper titles must appear in the transcript text
- If a name is ambiguous, note the ambiguity rather than guessing

When in doubt, leave it out.

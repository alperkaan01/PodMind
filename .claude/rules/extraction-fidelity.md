# Extraction Fidelity Rule

## Quotes
- Require: exact text, speaker name, timestamp (if available)
- Never truncate mid-sentence — include the complete thought
- If timestamp unavailable, set to null (do not guess)

## Insights
- Must be falsifiable statements, not vague summaries
- "X is important" is NOT an insight
- "X causes Y under condition Z" IS an insight
- Minimum quality bar: would a smart friend text you this?
- Each insight must stand alone without episode context

## People
- Must be mentioned by name in the transcript
- Include role/context only if stated in the episode

## Books
- Must be explicitly named in the transcript
- Include author only if stated in the episode
